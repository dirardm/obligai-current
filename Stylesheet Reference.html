Registry · scaling to 50+

Jurisdiction → regulation, in one control

A single combined control. Top row: jurisdictions, with the EU treated as one. Bottom row: the active regulations as coloured pills, plus an + Add button that opens a searchable popover scoped to whichever jurisdiction is active. Picking a jurisdiction filters both the pills and the popover.

One control, three jobs
The chips along the top scope everything below to one jurisdiction (EU counts as one). The coloured pills underneath show what's active and let you dismiss any of them with ×. The + Add regulation slot opens a searchable popover already filtered to the active jurisdiction — so you never scroll through 50 entries looking for the one you need.

The combined control · live

Jurisdiction
Regulations

Pick a jurisdiction (top row) to scope the pills and the + Add popover below. The EU is one jurisdiction; every other country is its own. Pills carry their registry colour. The × on a pill removes it from the active scope.

When to use which

Surface Primary control Notes
Global register Full combined control Jurisdiction chip row across the top, pills + Add below the table head.
Dashboard Pills row only (read-only) Inherits scope from the previous screen · chip row hidden.
Conflict graph Combined control, multi-select biased to 2–4 pills Users typically compare two regulations across two jurisdictions.
Detail page Pill row only (no chips, no Add) The pill reads as a header chip for the regulation being viewed.
Slide / report cover Static badges No interactivity needed.
